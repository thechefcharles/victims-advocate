=== CREATE: lib/server/organizations/inviteService.ts ===

Extract invite acceptance and creation logic from route handlers. Must export:

createOrgInvite(params: CreateInviteParams, ctx: AuthContext): Promise<InviteView>
  - Verify ctx passes can(ctx, 'org:invite', { orgId: ctx.orgId }) — throw FORBIDDEN if not.
  - Generate cryptographically random token, store token_hash = sha256(token).
  - Insert into org_invites: organization_id, email, org_role, token_hash, expires_at = now + 7 days, created_by = ctx.userId.
  - Return serializeInviteView(row).
  - Do NOT return the raw token in the view — only in the creation response envelope for email sending purposes (caller wraps it).

acceptOrgInvite(token: string, ctx: AuthContext): Promise<MemberView>
  - Hash token → lookup org_invites by token_hash.
  - Verify: row exists, used_at is null, revoked_at is null, expires_at > now(), email matches ctx user email.
  - Verify user does not already have active org_memberships row (check unique constraint before insert).
  - Insert org_memberships: { user_id: ctx.userId, organization_id: row.organization_id, org_role: row.org_role, status: 'active', created_by: ctx.userId }.
  - Call syncOrganizationLifecycleFromOwnership(row.organization_id).
  - Mark invite: used_at = now(), used_by = ctx.userId.
  - Return serializeMemberView(membershipRow).
  - Each step must throw typed errors (FORBIDDEN, NOT_FOUND, CONFLICT) — never return raw DB errors.

revokeOrgInvite(inviteId: string, ctx: AuthContext): Promise<void>
  - Verify can(ctx, 'org:revoke_invite', { orgId: ctx.orgId }).
  - Set revoked_at = now(), revoked_by = ctx.userId where id = inviteId AND organization_id = ctx.orgId.
  - Throw NOT_FOUND if no rows updated (prevents cross-org revocation).

listOrgInvites(orgId: string, ctx: AuthContext): Promise<InviteView[]>
  - Verify can(ctx, 'org:invite', { orgId }).
  - Query org_invites WHERE organization_id = orgId — always filter by orgId.
  - Return rows.map(serializeInviteView).
